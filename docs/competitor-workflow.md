# 竞品流程约束

对标 BurrisConnect / Eliminator 6 的核心不是调参页，而是固定流程：

1. 在 App 建立弹药 Profile：弹种、BC、弹道模型、初速、弹重、镜高、归零距离、SCI 最低末端能量。
2. App 通过 Bluetooth 将 Profile/DOPE 同步到瞄具。
3. 猎场中按测距键，瞄具读取距离、角度、环境。
4. HUD 显示距离、风偏、角度、环境、末速、末端能量。
5. 阵列点亮对应 holdover / windage 落点。
6. 如果测距点处能量低于用户阈值，分划中显示 X。

本 App 的页面骨架固定为：

- 顶部状态栏：BLE、当前 Profile、主要动作。
- 左侧流程：设备、弹药、环境、目标/SCI、HUD、同步。
- 中间配置：当前步骤表单。
- 右侧预览：HUD 黑框、目标、阵列落点、SCI X、解算结果。
- 手机端：同一流程变为底部导航，配置区单列，预览区优先显示。
